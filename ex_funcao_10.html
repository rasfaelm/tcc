Questão 10 UEL

Se uma função f, do primeiro grau, é tal que f(1) = 190 e f(50) = 2052, então f(20) é igual a:


Resposta:

Como foi dito que se trata de uma função do primeiro grau, ela pode ser descrita da seguinte forma:

f ( x ) = a ⋅ x + b

Além disso, temos os valores de 2 pontos que resolvem a função: (1; 190) e (50; 2052). Com isso, podemos formar um sistema de equações:

{ 190 = a ⋅ 1 + b 2052 = a ⋅ 50 + b

Multiplicando a primeira equação por (-1), temos:

{ − 190 = − a ⋅ 1 − b 2052 = a ⋅ 50 + b

Resolvendo o sistema, temos:

1862 = 49 ⋅ a

Portanto, a = 38. Com isso, podemos agora descobrir o valor de b substituindo o valor em qualquer uma das duas equações iniciais. Utilizaremos a primeira:

190 = 38 ⋅ 1 + b → b = 152

Agora, temos a forma geral da equação:

f ( x ) = 38 ⋅ x + 152

Como a questão pede para resolvermos f(20), substituiremos x na equação geral:

f ( 20 ) = 38 ⋅ 20 + 152 = 760 + 152 = 912
