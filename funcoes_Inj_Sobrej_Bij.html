Função injetora, sobrejetora e bijetora

As palavras "injetora", "sobrejetora" e "bijetora" são termos usados na teoria das funções para descrever diferentes tipos de relações entre os elementos do domínio e da imagem de uma função. Esses termos são frequentemente usados em contextos de matemática discreta e análise funcional. Vamos entender o significado de cada um deles:

Função Injetora (injetividade): Uma função é considerada injetora (ou injetiva) se cada elemento distinto do domínio é mapeado para um elemento distinto na imagem. Em outras palavras, a função não associa dois elementos diferentes do domínio ao mesmo elemento da imagem. Isso significa que não há "colisões" ou sobreposições entre os elementos do domínio e da imagem. Graficamente, uma função injetora não tem linhas horizontais que cruzam o gráfico mais de uma vez.

Matematicamente, uma função f é injetora se, para quaisquer dois elementos diferentes x1 ≠ x2 no domínio, f(x1) ≠ f(x2).

Função Sobrejetora (sobretividade): Uma função é considerada sobrejetora (ou sobrejetiva) se cada elemento no contradomínio (conjunto de possíveis valores de saída) é atingido por pelo menos um elemento do domínio. Em outras palavras, a função "cobre" todos os elementos do contradomínio.

Matematicamente, uma função f é sobrejetora se, para cada elemento y no contradomínio existe pelo menos um elemento x no domínio tal que f(x) = y.

Função Bijetora (bijetividade): Uma função é considerada bijetora (ou bijetiva) se ela é tanto injetora quanto sobrejetora. Isso significa que cada elemento único do domínio é mapeado para um único elemento na imagem, sem colisões, e que cada elemento no contradomínio é alcançado por pelo menos um elemento do domínio. Graficamente, uma função bijetora é uma correspondência um-a-um entre o domínio e o contradomínio.

Matematicamente, uma função f é bijetora se ela é injetora e sobrejetora ao mesmo tempo.

Em resumo:

  • Injetora: Nenhum par de elementos diferentes do domínio é mapeado para o mesmo elemento da imagem.

  • Sobrejetora: A função "cobre" todos os elementos da imagem.

  • Bijetora: Cada elemento único do domínio é mapeado para um único elemento na imagem, e cada elemento na imagem é alcançado por pelo menos um elemento do domínio.
